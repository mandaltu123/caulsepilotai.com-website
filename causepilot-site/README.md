# CausePilot AI Website

Modern AI consulting and product engineering landing page built with Next.js, React, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
cd /Users/tsm/work/caulsepilotai.com/code/caulsepilotai.com-website/causepilot-site
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production Build

```bash
npm run build
npm start
```
